# Sistema de Gestão de Funcionários em Python - V2.0

Projeto desenvolvido para consolidar conceitos fundamentais de Python com foco em organização de código e lógica de negócios.

## Funcionalidades
- Cadastro de funcionários com ID automático
- Listagem detalhada
- Atualização de salário
- Remoção de funcionários
- Cálculo de média salarial
- Relatório geral
- Persistência de dados em arquivo

## Conceitos aplicados
- Programação Orientada a Objetos
- Funções e módulos
- Separação de responsabilidades
- Leitura e escrita em arquivos
- Tratamento de exceções
- Função main()

## Tecnologia
- Python 3
